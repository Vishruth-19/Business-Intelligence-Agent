# Business-Intelligence-Agent
An AI-powered business intelligence agent that integrates with monday.com to provide founders and executives with instant, actionable insights from sales pipeline and operational data.


Key Features

Conversational Interface: Natural language querying for complex business data.

Monday.com Integration: Dynamic fetching of "Deals" and "Work Orders" boards.

Data Resilience: Automated cleaning of inconsistent date formats, currency symbols, and null values.

Intelligent Analysis: Uses GPT-OSS-120b (via OpenRouter) to generate and execute Python code for high-speed, accurate mathematical calculations.

Leadership Summaries: One-click generation of executive-style briefings.


Tech Stack

Frontend: Streamlit.

Language Model: GPT-OSS-120b via OpenRouter.

Data Processing: Pandas.

Integration: monday.com GraphQL API.
